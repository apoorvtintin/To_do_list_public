# to_do_list
A distributed to do list

---

**To compile:**

`make format; make clean; make;`

---

**To start the server:**

`./server {IP} {Port}`

---

**To start the client:**

`./client {IP} {Port} {heartbeat interval}`

The interval is optional. If no value is given as input then default 10 sec value is set.
